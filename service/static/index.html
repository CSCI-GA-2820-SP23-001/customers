Customers REST API Service

GET /customers to see all the customers.
GET /customers?first_name=john to see all of the customers with the first name "john".
GET /customers/{id} retrieves a customer from the DB using an ID.
PUT /customers/{id} updates a customer using an ID.
DELETE /customers/{id} removes a customer from the DB using an ID.
POST /customers creates an new customer.

Example JSON Data:

{"first_name": "Abraham", "last_name":"Abrahamson", "email":"aabrahamson@honest.com", "password": "th3honest1"}
